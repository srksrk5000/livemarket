\# NSE AI Momentum Screener



A fully automated, mobile-first NSE stock screener optimized for Android flagship devices like Samsung S25 Ultra.



---



\# Features



\- Real-time NSE stock scanning

\- Yahoo Finance live ingestion

\- Anti-freeze async queue engine

\- RSI calculations

\- 20-day breakout detection

\- Institutional momentum scoring

\- ORB intraday scanner

\- VWAP + OBV analysis

\- Abnormal volume radar

\- Mobile-first responsive UI

\- Dark professional fintech design

\- Custom ticker analyzer

\- Auto sorting by Score + Upside



---



\# Tech Stack



\- React + Vite

\- TailwindCSS

\- Lucide React Icons

\- Yahoo Finance APIs

\- Async Batch Processing Engine
